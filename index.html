HTML Forms Cheat Sheat

Author: Pat McGroin

Form ElementDescription
<form>Used to collect and send information to an external source
<input>Used to render a variety of input fields on a webpage including text fields, check buttons etc
<label>Provides identification for a specific input based on matching value's of the <input>'s id attribute and for attribute of the <label>
<select>Used to create a drop down list with choices using one or more <option> element
<datalist>used as a basic search/autocomplete functionality paired with an <input> element for users
<textarea>Used to create a textbox for multi-line input
For more info about other form elements, please visit the mozilla documentation
